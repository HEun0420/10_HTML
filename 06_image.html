이미지 관련태그

src 속성에 경로를 설정하여 이미지 파일을 불러온다.

자주 사용하는 속성들

시진의 경로가 잘못되거나, 이미지를 제대로 표현할 수 없을 경우 대체 텍스트 용도로 사용된다.


width, height 속성

    사진의 높이와 너비를 지정할 수 있다
    고정된 단위의 크기로도 지정 가능하며
    가변크기 단위의 크기로도 지정이 가능하다.

고정 크기 단위 : 화면 사이즈가 줄어도 크기가 변하지 않는다.

가변 크기 단위: 화면 사이즈 혹은 기준에 따라 크기가 변경된다.
